Russian rock bands

and one band in particular

Attempting to provide an exact count of Russian rock bands can be a daunting task due to their numerous nature. However, this particular project endeavors to provide an approximate estimation of the existing bands, shedding light on their collective landscape. Moreover, the project offers a comprehensive overview of these bands, with a particular focus on DDT, which holds a special place as the project's personal favorite. Within the project, one can explore the intricate composition of the DDT band, as well as delve into the diverse range of musical genres in which they skillfully create their music.

Methodology, tools and data sources

Before starting to write the queries for the project, we explored the Polifonia corpus and tried writing sentences like Composition of Russian rock bands or Composition of Romanian music bands on FRED (Text-to-AMR-to-FRED (cnr.it)) to get our heads around the terminology we need to use when we create stories about Russian and Romanian bands. We decided to use Wikidata when creating our stories as we consider it useful for future application in our careers, thus we found the terms and Wikidata codes on FRED and after studying in detail the queries we looked at in class and practiced during the hands-on lesson, we could use them to write our own queries on Wikidata Query Service.

For the queries we used SPARQL language that is based on triple patterns, including subject, object and predicate. To create a query we used SELECT (to define the information we are looking for), WHERE (to define the source of information), OPTIONAL (for example, to add pictures for bands when there are some but not restrict the search in case there is no picture available), LIMIT (to restrict the search and limit it to a particular number of results â€“ not to overwhelm the graphs).

To check if the query was written in the right way, we ran the query on Wikidata Query Service before inserting it into Melody. But once the query worked, we created a linechart/pie chart/table on Melody to show the results of the search in a more beautiful and understandable way.

Challenges

While writing queries on Wikidata Query Service, we encountered a few challenges such as lack of information (for example about some of the Romanian rock bands), in which case we changed the focus of our research to Romanian music bands in general. Also, another challenge for us was creating a map , showing Russian rock bands and Romanian music bands according to the city where they were formed, on Melody. We used the ?formationLocationLabel and wrote the query that worked on Wikidata Query Service, however the result was 0, that is why we could not display it on Melody.
